vps-init
========

vps initial install script written in bash

**usage:** `bash <(curl http://goo.gl/D3sdsn)`

* updates apt repos

* installs 

  * vim
  * git
  * apache2
  * mysql
  * php
  * some php mods
  * composer

you can just get the file at url https://raw.githubusercontent.com/emreyo/vps-init/master/vpsi.sh with curl and execute
